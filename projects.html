Pacman Game

Language: Java




Developed a Pacman game that includes a pacman and ghosts with the adaptation to Pacman game concept by focusing on Object Oriented Concepts, GUI and various methods. It also includes a scoreboard that tracked the real time, player name and score. The score increased by 1 as the pacman touch each circle along the maze. When the pacman touched a ghost(s) the user can resume the game 3 times until it is Game Over.

Social Circles Twitter

Language: Rust




Analysis of Twitter dataset mainly focusing on Six Degree of Separation with Rust Programming on Twitter Dataset from Stanford Network Analysis Database.
